# Q4 - Users-API

## Technology
Node.js - Express.js

## How to Start
1. `npm install`
2. `npm start`
3. Use [Postman](https://www.postman.com/) to view APIs on `http://localhost:5000/`

## Data
All users stored in `/users.json`

## APIs
### Path `/`
- `GET` - get all users data
- `POST` - create a user data (json-body: `{firstname: "", lastname: "", birthday: ""}`)
